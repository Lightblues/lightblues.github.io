安装 Windows

微 PE 安装

用的是微 PE，相对来说最为省心的一种安装方式了

  • 若是第一次安装建议全文阅读 http://www.wepe.com.cn/ubook/，没啥好说的，但若是第一次尝试的话总会心慌慌；

下载源&版本问题

最推荐的自然是从官网下载，不过有速度的问题；这就涉及了第三方的下载，不推荐直接从不明的百度云等渠道下载，自己在用 https://msdn.itellyou.cn/

  • 其实困扰最多应该是 win 的版本问题，教育版不要装了，但其他专业版或是企业版等的问题就众说纷纭了
  • 之前用过专业版，可能是来源的问题不是很纯净，系统体验一般，例如说自带应用商店等，会带一些基本的软件，例如自带的 Onenote；
  • 现在用的是企业版，直观感受就是很自由，然后也没有应用商店、Onenote 等，很纯净
  • 「操作系统-Windows 10 Enterprise LTSC 2019」，选择最新的相应版本即可，我用的是 「Windows 10 Enterprise LTSC 2019 (x86) - DVD (Chinese-Simplified)」

关闭 Windows Defender

由于某些原因，还是需要关闭 Windows Defender 的， Win+R 中输入 gpedit.msc 打开「本地组策略编辑器」，在 「计算机配置-管理模板-Windows组件-Windows Defender 防病毒程序」中（右侧）找到「关闭 Windows Defender 防病毒程序」，双击，设置为 「已启用」，确定即可。

设置开机启动

  • 简言之，就是打开 C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp，把想要的软件创建一个快捷方式即可。

开机启动程序如何设置

  • 事实上可以设置注册表（可惜我在 Enterprise 版本中没找到 Start Menu），仅做参考

    将需要开机自动启动的软件的快捷方式粘贴到“系统启动文件夹”里，位置为 C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp 。

Uninstall Tool

  • 一个好用到我需要放在这一部分记录的软件，简洁、高效、干净；
  • Portable 版本，单独放在一个文件夹，「固定到『开始』菜单」方便使用；

Windows 相关设置

这部分不过是个人审美和习惯

  • 关于背景，MacOS 下有一张非常爱的。这里 最后一张，配合 Retina 屏的质感真的很棒；不过只有 355K 的照片放在 27 寸的 4K 屏上像素不是很好，显示效果也一般，遂放弃；
  • 触摸板：三指点击设为鼠标中键【在台式上就不存在的了】
  • 添加 This PC 到桌面： Settings-Personalisation-Themes-Desktop icon settings

安装 Microsoft Store

专业版在很纯净的同时，附带的一个烦恼就比如是没有应用商店；当然自己几乎也不用，但是自己日常使用频率很高一个软件就是 OneNote；当然，下面说的 Office 套餐里带有 OneNote2016，也是最后一版了（之后的全部内置到到 win10 应用商店，并且强制关联 OneDrive。

不可否认当时应用版 OneNote 的强大之处，但也带来了过于臃肿的毛病。1. 软件卡顿，使用体验不流畅；2. 比如说自己遇到很多次开启时要等很久「清理上次关闭内容」的菜单，并且往往是直接卡死；3. 界面一体化比较差，网上说是诈尸更了下黑色模式，可以自己开了之后发现仅仅是边框黑色而页面还是白色；4. 另外短暂用了一下似乎同步也不太行。

于是，最终放弃 OneNote2016 怀抱 win10 版 OneNote

  • 可以参考这篇文章 Windows LTSC、LTSB、Server 安装 Windows Store 应用商店 ，思路就是下载安装包之后 CMD 安装；然而自己失败了，原因是这些包的名字真的傻傻分不清楚；
  • 于是找到了下面的参考链接 LTSC-Add-MicrosoftStore ，集成了比较全面的依赖项和安装包，并且直接写了个 cmd 文件；运行过程中报了写错，不过发现程序部分已经找到 Micorsoftt Store 了，于是没有在意，成功安装；
  • 搜索安装 OneNote 即可；最后，强迫症还是要删除 OneNote2016 的，这里又出了点波折，想看的见下面 软件部分-Office套餐；
  • 此外，除了上面的方法，还看到了 Windows10 LTSB/LTSC版安装应用商店与UWP 这篇文章，思路是从和专业版版本号一致的普通版中爬到相关软件，之后手动安装，思路更为清晰；然而这样做的问题在于涉及反复的文件权限操作，更为复杂。
  • 另外，关于 MicrosoftStore 下载速度很慢的问题，参考知乎的一个帖子找到了 UsbEAm Hosts Editor [多平台hosts修改] V3.50 这个软件，不怎么会用随便试了下（因为自己也不用）；后来发现下载的速度是可以升上去的，从原来的几十 K 可以到几十 M，时间消耗最多的倒是之前的定位和检测过程，不知道是不是经过修改之后的效果。

网络连接 & 下载/传输

科学上网

  • SSR：可以参考这个项目 https://github.com/qingshuisiyuan/electron-ssr-backup；另外，今天才发现 SSR 右键-选项 是可以直接设置为「开机启动」的，不需要像上面讲到的那样自行设置了。

浏览器：Chrome

  • 浏览器已经习惯了 Chrome 了（可能自己的隐私意识还没有那么强吧），直接下载安装即可，当然科学上网之后下载才比较快——以及更为重要的登录和同步。

关于在 win 上使用 iCloud

这个问题再用了一段时间的 MBP 之后就显得有点紧迫了：苹果家的生态绑架有点可怕。

  • 少数派的 多种方法让你在Windows上优雅的使用iCloud服务 这篇文章给出了 3 年前的一个完备的解决方案，不过现在有点落伍了；
  • 核心的解决方案就是要把网页编程独立的软件，目前的 Chrome 可以比较轻松的做到：「更多-更多工具-创建快捷方式」，具体可参见 https://zhuanlan.zhihu.com/p/59692068

下载工具

  • 当然是 IDM 啦，win 下绝对的神器；

共享文件夹

  • 事实上这个折腾了挺久的，虽然之前折腾过 NAS 但似乎没啥用，发现一个比较完善的参考 Mac OS X 访问 Windows共享文件夹 ；
  • 还是挺复杂的，不过一步一步来就可以了；采用 SMB 连接；
  • 最后一步中「一般需要输入Windows对应用户的名称和密码」，指的是 Microsoft 账户，即用户名是账号邮箱（而非电脑登录时 Administrator 和 PIN）加上密码（吐槽一下微软家的密码还必须包含大小写）；【顺带一提，若需要添加账户的话，是在「设置-账户-家庭和其他应用」中添加的 Microsoft 账户，从这一点似乎可以看出来 Win 是针对单用户而非 Linux 逻辑的？】

Mac 远程控制 Windows

考虑到 MBP 的便携性，之前在学校也是只带 MBP 而把 Surface 留在寝室的；考虑到买的是乞丐版 Mac，内存和硬盘的原因多需要其他电脑的配合。

  • 试过了 TeamViewer，发现更为好用的是 MS Remote Desktop 。（注意，这款 APP 国区没有）

记录&Markdown

Typora+PicGo

  • 可以参考 PicGo + Gitee(码云)实现markdown图床 进行配置；
  • Typora：最爱的 Markdown 软件了，除了文档管理外几乎满足了我对于一个个人记录软件的想象。当然需要一定的个性设置才会用的更舒服；
  • 相关设置可以另外开一篇了，这里将现阶段对自己最重要的（也最麻烦的）部分：图片处理；
  • 之前在这里纠结了很久走了些坎坷，总结一下。Typora 是默认支持自动上传到图床的，Mac 上有一个开源项目 uPIC，而在 Win 则是 PicGo（其实 Mac 也有之前用的就是这个，不过因为要安装 Gitee 的插件安不上所以转而用起了 uPIC；
  • 设置-图像 部分设置为 自动上传，在 Typora 上的部分是比较直观的，下面选择 PicGo；
  • 然后是安装 PicGo，默认不支持 Gitee 需要需要在「插件」部分搜索 gitee 安装对应的两个，注意之前要装好 Node.js，上官网下载即可；这部分可能是因为网络部分不太稳定，开关机了几次还安好，相关配置可以查文档

软件部分

  • 还有一个问题是用「安装版」还是「便携版」，两者的区别没有深究过，不过想来包括：
    • 便携版若需要管理员权限（例如我常用的 Everything）并且是经常要用的，那么每次使用需要进行确认，影响使用体验（所以这种情况下一般用安装版比较好）；
    • 安装版的配置等迁移会比较麻烦，而便携版直接复制粘贴即可，对于一些配置很繁复或者经常要变动的软件来说，倾向于用便携版；
  • 我目前是把便携版软件都单独放在了一个文件夹中，对于一些必要的软件设置开机启动；

输入法

更新：已用回原生。

原生的输入法有着很棒的输入体验，包括：１. 响应速度；２.配色很赞．但有两个致命的问题：１. 全半角极为死板，在中文下按 shift 切换为英文会出现奇怪的全角英文字母，这无法忍；2. 字典和词汇学习能力极弱。

之前还倔强过一段时间Google输入法（至今仍保留着其安装包），但信仰自己也放弃了这个产品，年久失修的东西用起来毕竟有些不顺手。遂投入了搜狗的怀抱。

  • 搜狗：由于一些输入习惯的原因，win 自带的这里就需要很多设置了。还是使用了搜狗，用的是微信的账号，以便同步词库和设置。

    设置部分包括：程序-设置-常用 部分，记得设置为「半角」。Orz 看了下相较于 Mac 版本的差别有点大啊，很多功能都没有了，只好曲线救国。（由于之前设置过部分，有些遗忘，以下是并不完全罗列）

    • 考虑到搜狗在 win 版本上的状况，记得把一些无关功能全部禁用；
    • 中英文切换还是熟悉的 Shift ；
    • 居然找不到中英文之间插入空格的选项，只好手动输入，还好问题不大；
    • 标点符号可以在「常用-标点符号-自定义标点符号」中进行设置，而且更为细化，目前配置
      • '第一次 为 「，'第二次 为 」 ，以实现自动匹配；
      • 左上角的那个 点号 设置为 Markdown 行内行代码符号（即英文状态下的默认输入）；
    • 记得把浏览器那个搜狗联想给关了，「设置-搜索」选项，或者直接在浏览器输入的时候到鼠标到下面的选框进行设置；
